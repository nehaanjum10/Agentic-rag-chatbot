# 🤖 Agentic RAG Chatbot

Powered by LangChain + Groq + FAISS + LLaMA 3.3

## Features
- 📄 PDF Upload & Processing
- 🔍 Smart Query Routing
- 🌐 Web Search Integration
- 🤔 Self Reflection
- ✨ LLaMA 3.3 Answers
- 💬 Chat History

## Tech Stack
- LangChain
- Groq API (LLaMA 3.3)
- FAISS Vector Store
- FastEmbed Embeddings
- Streamlit UI

## Setup
1. Clone repo
2. Install requirements
   pip install -r requirements.txt
3. Add GROQ_API_KEY to .env
4. Run streamlit run app.py

## Usage
1. Upload any PDF
2. Click Process PDFs
3. Ask questions!
